# Soluciones a los enunciados de declaracion de variables con val y var

## Enunciado 1

Declara una variable que almacene el número de habitantes de una ciudad.

<details>
  <summary>Solucion</summary>
  
  ```kotlin
  var numeroHabitantes:Double = 341,890
  ```
</details>
